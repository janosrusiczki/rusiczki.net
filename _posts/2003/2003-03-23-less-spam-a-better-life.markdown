Read in a [post by Kasia](http://cygnus.unix-girl.com/blog/archives/000843.html "Kasia's Record is 41.9 points!!!") about [Spam Assassin](http://www.spamassassin.org/) which lead me to [SAProxy for Windows](http://saproxy.bloomba.com/), a POP3 proxy server for Windows which will tag every SPAM mail so that your mail program can filter them out easily or put them directly in the Trash folder.

This is not only a useful piece of software but it's also quite funny as it rates every mail according to the results of the spam tests it does on each mail. So you can go through all your spam and see which rates higher. Geeky. :-)
